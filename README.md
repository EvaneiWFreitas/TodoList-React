📋 Todo List em React

Projeto completo de uma aplicação Todo List (Lista de Tarefas) desenvolvido utilizando React, com foco em organização de tarefas, produtividade e boas práticas de desenvolvimento front-end.

🚀 Sobre o Projeto

Esta aplicação permite ao usuário gerenciar suas tarefas diárias de forma simples e eficiente. É possível adicionar, editar, concluir e remover tarefas, proporcionando uma experiência prática para o dia a dia.

O projeto foi desenvolvido com o objetivo de aprimorar habilidades em React, manipulação de estado e criação de interfaces modernas.

🛠️ Tecnologias Utilizadas
⚛️ React
📜 JavaScript (ES6+)
🎨 CSS
📦 Node.js (ambiente de execução)
📁 HTML5
✨ Funcionalidades
✅ Adicionar novas tarefas
✏️ Editar tarefas existentes
✔️ Marcar tarefas como concluídas
❌ Remover tarefas
🔍 Filtro de tarefas (todas, concluídas, pendentes)
💾 Persistência de dados (ex: LocalStorage)
